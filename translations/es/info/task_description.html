Hay un polígono convexo en un plano de coordenadas. Este polígono se presenta como una lista de las coordenadas de sus vértices. Cada vértice está conectado secuencialmente, considerando además que el ultimo se conecta en el primero. El polígono tiene N vértices. Deberás escribir un programa que calcule el área del polígono., teniendo en cuenta que el resultado tenga una precisión de ± 0.1.

Datos de Entrada: Coordenadas como una lista de listas. Cada lista contiene dos enteros (int).

Output: El área del polígono como un decimal (float).

Ejemplo:

checkio([[1, 1], [9, 9], [9, 1]]) == 32
checkio([[4, 10], [7, 1], [1, 4]]) == 22.5
checkio([[1, 2], [3, 8], [9, 8], [7, 1]]) == 40
checkio([[3, 3], [2, 7], [5, 9], [8, 7], [7, 3]]) == 26
checkio([[7, 2], [3, 2], [1, 5],
         [3, 9], [7, 9], [9, 6]]) == 42
checkio([[4, 1], [3, 4], [3, 7], [4, 8],
         [7, 9], [9, 6], [7, 1]]) == 35.5

¿Cómo se usa?: Este concepto es muy útil en topografía y arquitectura. Puedes mejorar el algoritmo y calcular la superficie de cualquier zona en un mapa, teniendo en cuenta la pendiente y otros puntos de interés. También podrías utilizar esta idea para construir algo como un edificio, o un barco.

Condiciones: 3 ≤ N ≤ 8
∀ x, y ∈ coordenadas: 0 ≤ x ≤ 10; 0 ≤ y ≤ 10;
