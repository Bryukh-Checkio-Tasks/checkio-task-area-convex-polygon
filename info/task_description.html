There is a convex polygon on a coordinate plane. This polygon is presented as a list of vertices' coordinates. Each vertex is connected sequentially with the last connecting back to the first. A polygon has N vertices, where 3 ≤ N ≤ 8. You should write a program that will calculate the area of a polygon. The margin of error for the result should be +/- 0.1. CheckiO is aware of the float comparison problem and the editor will check the result as +/- 0.1.

Input: A list of lists. Each list contains two integers.

Output: A float. The measurement error is 0.1.

Example:

checkio([[1, 1], [9, 9], [9, 1]]) == 32
checkio([[4, 10], [7, 1], [1, 4]]) == 22.5
checkio([[1, 2], [3, 8], [9, 8], [7, 1]]) == 40
checkio([[3, 3], [2, 7], [5, 9], [8, 7], [7, 3]]) == 26
checkio([[7, 2], [3, 2], [1, 5],
         [3, 9], [7, 9], [9, 6]]) == 42
checkio([[4, 1], [3, 4], [3, 7], [4, 8],
         [7, 9], [9, 6], [7, 1]]) == 35.5

How it is used: This task is useful for understanding image recognition systems.
